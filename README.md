# Proyecto-Individual Daniela Prado Vargas C26070
Este proyecto analiza los datos del Wellcome Global Monitor 2020 para estudiar la percepción pública sobre la ciencia, el gobierno y las instituciones de salud en América. Utilizando R, se exploran tendencias de confianza, el impacto económico del Covid-19 en los hogares y las actitudes hacia el gasto gubernamental. Se compara la relación entre el apoyo a la ciencia y el rendimiento percibido de los gobiernos en América Latina y Norteamérica.
# Objetivo
Analizar las percepciones públicas sobre la ciencia, el gobierno y las instituciones de salud en América durante el Covid-19, explorando las diferencias en confianza, el impacto económico en los hogares y las actitudes hacia el gasto gubernamental en la prevención y cura de enfermedades.
